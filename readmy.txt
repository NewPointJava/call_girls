Этот бот предназначен для создания, мониторнга, управления заказами на уборку.



Перед началом работы нужно установить id Администраторов и Клинеров.

Для того чтобы узнать id - воспольуйтесь ботом https://t.me/getmy_idbot

(Находясь в аккаунте администратора/клинера перейдите в бота https://t.me/getmy_idbot и  напишите /start. Скопируйте ваш ID )

Зайдите в папку jsons_config , и откройте файл settings.json в любом тектовом редакторе.
Напротив поля - admins - нужно вписать через запятую id всех администраторов.
Пример "admins": [694443138, 444908023]    Если администратор 1 - то только одно число без запятых.

Установка id Клинеров происходит так же, но поле называется "cleaners"
Пример "cleaners": [694443138, 444908023]

Так же нужно установить зависимости для python
используйте команду в терминале
pip install -r requirements.txt

Функционал

Пользователь:
    Может создавать заказ на уборку
    Просматривать отзывы
    Оставлять отзывы завершенных заказов  (Возможно только когда клинер пометил заказ выполненым)


Клинер:
    Просмотр графика работы
    Просмотр каждого отдельного заказа
    Помечать выполненые заказы


Администраторы:
    Могут самостоятельно создавать заказ и сразу назначать клинера. (Пока что только в полуавтоматическом режиме)
    Принять/удалить присланный заказа от пользователя (возможность личного контакта с ним)
    Могут просматривать список всех не обработанных заказов
